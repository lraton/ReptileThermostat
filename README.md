
# Terrarium Thermostat

A thermostat for terrarium made with arduino and 3d printed case
![profile pic](/img/large_display_thermostat_1.jpg)


Component 1x:
- Arduino
- DHT11 
- RELE 
- I2C Display
- Buzzer
- Led
- Encoder whit push button
- Clock module

Extra parts:
- Insert nut m2/m4
- Screw m2/m4
